練習django開發，這次為了一些平常開發的情形去做考量，決定做此架構。

- **分支管理**：git 有 dev 與 master，當需要改的時候會切 feature 分支進行修改  
- **開發架構**：vagrant + ubuntu + docker + django + postgresql + vue，做一串連開發練習
- **README_DAILY.txt** - 日常開發需要執行的指令（例如：啟動 Django 開發伺服器、執行資料庫遷移等）
- **README_SETUP.txt** - 第一次安裝的步驟（例如：環境設置、依賴安裝等）
- **第一次git建立需知.txt** - Git 基本指令（例如：如何初始化倉庫、分支操作等)，基本上都會了可略過。
